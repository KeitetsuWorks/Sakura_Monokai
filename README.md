Monokai for Sakura Editor
========

## Description
サクラエディタをMonokai風の配色するための色設定ファイルと，C/C++とVerilogのソースコードをシンタックスハイライト表示するためのキーワード設定ファイルです．

## Requirement
* [サクラエディタ](http://sakura-editor.sourceforge.net/)
* [MigMix 1M](http://mix-mplus-ipa.osdn.jp/migmix/)
