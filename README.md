Monokai for Sakura Editor
========

## Description
サクラエディタをMonokai風の配色するための色設定ファイルと，下記の言語のソースコードをシンタックスハイライト表示するためのキーワード設定ファイルです．

* C/C++
* Tcl
* Verilog
* Markdown
